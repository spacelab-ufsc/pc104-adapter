\chapter{Hardware Overview} \label{ch:hardware}

Starting from hardware version 3.0, the project uses a single PCB design that can be assembled either as a top or bottom board. The final configuration depends exclusively on the type and orientation of the PC-104 connectors populated during assembly. This approach simplifies manufacturing, inventory management, and maintenance while preserving full mechanical and electrical compatibility. The PCB was designed using KiCad v7.0.9-1 \cite{kicad}.

All pins of the PC-104 connector are routed between the two boards, this way, this adapter can be used in any configuration of the PC-104 pins.

The board also have an inner cavity to allow vertically passing internal cables of the satellite. All the screw holes are connected to the copper planes of the board, but isolated from the PC-104 tracks (in other words, no signal is grounded to the chassi).
